Ecommerce platform using stripe API for payment integration

Tech stack

React, Node.js, Express, MongoDB, JavaScript, Redux, JWT, Stripe
  • Live Demo
  • Source code

This is another Full-Stack MERN application, which aims to create a shopping website like Amazon or other e-commerce websites.

  • Utilized JSX/Styled-Components/Material-UI, React.js to design the front-end UI of an e-commerce website.
  • Built Restful APIs with Node.js(express)/MongoDB, handled HTTP requests and responses, created CRUD operations to fetch data with queries.
  • Utilized Redux Toolkit to manage application states and asynchronous redux actions.
  • Implemented JWT to create user authentication, improve security and verify requests.
