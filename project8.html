Amusement Park Game

Tech stack

Unity, C#, VR
  • Live Demo
  • Source code

This is a 3D virtual amusement park designed with Unity. The game allows users to ride a roller coaster or a Ferris wheel, users are also able to enjoy a VR mode on their phones.

  • Designed a 3D virtual amusement park with Dinosaur theme using Unity3D.
  • Embedded a program which allowed users to ride a roller coaster or ferris wheel within the game.
  • Created a Day and Night feature to present a first-person and third-person view for the user experience
