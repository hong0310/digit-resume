Meal Recipe Finder App

Tech stack

HTML, CSS, JavaScript, API, Material-UI
  • Live Demo
  • Source code

This is a meal recipe finder application using API. Users are able to find according meal recipes by simply entering the key ingredient. There is also a YouTube video attached to each recipe to better help people.

  • Created a website for meal recipe finds for users to access meal recipes by entering key ingredients.
  • Implemented front-end App UI design with HTML/CSS.
  • Utilized React.js, React Router and Material-UI to build website interface.
